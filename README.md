Website statis interaktif dan berkinerja tinggi untuk wadah informasi dan publikasi Komisariat HMO "TRITON" ITB (Kabinet Daya Biru). 

## 🎯 Fitur Utama
* **Modern UI/UX:** Desain antarmuka *full-screen* dengan tema *Deep Ocean* dan elemen kaca (*glassmorphism*).
* **Interactive Animations:** Menggunakan *Liquid Wave background* dan *Scroll-Responsive SVG Curves*.
* **Responsive Design:** Optimal diakses melalui Desktop, Tablet, maupun *Smartphone*.
* **Arsitektur Jamstack:** Performa maksimal tanpa database tradisional, sangat aman, dan siap diintegrasikan dengan Headless CMS (Decap CMS) ke depannya.
* **Organogram Dinamis:** Profil interaktif untuk Badan Pengurus dan Program Unggulan.

## 🛠️ Teknologi yang Digunakan
* **Frontend Framework:** React 18
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion & Native CSS Keyframes
* **Icons:** Lucide React

## 📂 Struktur Aset (Penting untuk Kontributor)
Semua aset gambar diletakkan di dalam folder `public/assets/` dengan struktur ketat berikut:
- `/logos/` : Logo himpunan (`logo-hmo.jpg`) dan Kabinet (`Daya-Biru.png`).
- `/divisions/` : Ikon/logo tiap bidang (cth: `ketua-komisariat.png`).
- `/pengurus/` : Foto pengurus (cth: `foto-ketua.jpg`).
- `/news/` : Gambar *thumbnail* untuk kegiatan himpunan.
